# Vendor Comparison Matrix

## Metadata
- **ID**: vendor-comparison-matrix
- **Category**: Operations/Legal
- **Time Saved**: 3-6 hours
- **Recommended Model**: Any

## Description
Create structured vendor evaluation matrices with weighted criteria, scoring, and recommendation frameworks.

This skill generates comprehensive vendor comparison analyses that support procurement decisions. It creates weighted evaluation criteria, objective scoring frameworks, risk assessments, and clear recommendations for vendor selection.

## What You Get
- Comparison Matrix
- Weighted Scoring
- Risk Assessment
- TCO Analysis
- Recommendation
- Decision Framework

## Description
Create personalized learning paths with resources and milestones for skill development.

Generate comprehensive skill development plans including learning resources, practice projects, milestones, and assessment criteria for professional growth.

## What You Get
Complete learning path with curated resources, practice projects, and progress milestones

| Field | Type | Required | Description |

| Current Level | select | Yes | Beginner (No experience) / Intermediate (Some experience) / Advanced (Proficient) / Expert (Looking to specialize) |

| Time Available | select | No | 5 hours/week / 10 hours/week / 20 hours/week / Full-time |
| Preferred Learning Style | select | No | Video courses / Books/Articles / Hands-on projects / Mixed approach |

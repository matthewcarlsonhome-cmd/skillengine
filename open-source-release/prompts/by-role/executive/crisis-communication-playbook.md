## Description
Create crisis communication plans with messaging templates and stakeholder strategies.

Generate comprehensive crisis communication playbooks including situation assessment, stakeholder messaging, media response templates, and escalation procedures.

2. **Transparency** - Honest, factual communication
3. **Empathy** - Acknowledge impact on stakeholders
4. **Accountability** - Take appropriate responsibility
5. **Action** - Show concrete steps being taken

**Communication Hierarchy**:
1. Affected individuals (safety first)
2. Employees
3. Board/Investors
4. Customers
5. Media/Public
6. Regulators

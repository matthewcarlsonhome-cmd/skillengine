# RFP Response Generator

## Metadata
- **ID**: rfp-response-generator
- **Category**: Comprehensive Executive Communications
- **Time Saved**: 8-15 hours per RFP response
- **Recommended Model**: Any

## Description
Create compelling RFP/RFI responses with structured proposals and differentiators.

Generate professional RFP responses including executive summaries, capability statements, pricing frameworks, and win themes tailored to buyer requirements.

## What You Get
Complete RFP response framework with executive summary, technical response, and pricing guidance

## Inputs
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| RFP Summary / Requirements | textarea | Yes | Summarize the RFP requirements, evaluation criteria, and key asks... |
| Your Company / Solution | textarea | Yes | Describe your solution, capabilities, and relevant experience... |
| Key Differentiators | textarea | Yes | What makes you unique? Competitive advantages, special capabilities... |
| Known Competitors | textarea | No | Who else is bidding? Their strengths/weaknesses... |
| Deal Size | select | No | Small (<$100K) / Medium ($100K-$1M) / Large ($1M-$10M) / Enterprise (>$10M) |

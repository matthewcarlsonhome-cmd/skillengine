# Role Transition Playbook

## Metadata
- **ID**: role-transition-playbook
- **Category**: Comprehensive Executive Communications
- **Time Saved**: 4-8 hours per transition plan
- **Recommended Model**: Any

## Description
Create comprehensive playbooks for role transitions and knowledge transfer.

Generate structured role transition plans including knowledge transfer schedules, stakeholder introductions, critical information handoff, and success criteria.

## What You Get
Complete transition playbook with knowledge transfer plan, stakeholder map, and 30-60-90 day roadmap

## Inputs
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| Transition Type | select | Yes | Internal Promotion / Lateral Move / External Hire Onboarding / Outgoing Employee Handoff / Role Expansion |
| Role Description | textarea | Yes | Describe the role: responsibilities, team, key relationships, success metrics... |
| Critical Knowledge Areas | textarea | Yes | What knowledge must be transferred? Systems, processes, relationships, history... |
| Transition Timeline | select | No | 2 weeks / 30 days / 60 days / 90 days |
| Additional Context | textarea | No | Urgency, special circumstances, upcoming projects, team dynamics... |
